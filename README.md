# py3d

Нужен модуль numpy

pip install numpy

Оно по идее работает, но не доконца правильно и ни всё что нужно.

Пока можно подгузить заготовленную модельку в формате stl. Изменить её можно в файле py3d.py,
его же нужно и запускать.

Сверху шесть кнопок управляют перемещением камерой, снизу работают кнопки поворота влево, вправо
и кнопки сброса базиса камеры.

Красной точкой отмечен источник света.

Если смотреть под нужным углом, то кажется что всё правильно работает.

Всё завязано на numpy(абсалютно), поэтому тесты на энитаске не работают, но на компе должно работать.

Надеюсь хватит чтобы с распределённым хранилищем в сумме вышло +11 баллов :)
